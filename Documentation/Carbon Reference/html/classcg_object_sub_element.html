cgObjectSubElement Class Referenceabstract

Base class from which all object sub-element types derive. A sub-element represents any editable property of an object which might include (for instance) faces of a mesh, collision shapes for a rigid body object, etc. It essentially gives each object type the ability to expose parts of itself to the engine / editor such that it can be manipulated. Detailed Description

Base class from which all object sub-element types derive. A sub-element represents any editable property of an object which might include (for instance) faces of a mesh, collision shapes for a rigid body object, etc. It essentially gives each object type the ability to expose parts of itself to the engine / editor such that it can be manipulated.
