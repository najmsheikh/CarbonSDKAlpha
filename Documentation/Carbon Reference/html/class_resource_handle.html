Detailed Description

template<class _RT, class _BaseClass = ResourceHandleBase>
class ResourceHandle< _RT, _BaseClass >

This class is similar to a smart pointer which manages a single reference to an individual resource and handles all reference counting. This class is derived from the base 'ResourceHandleBase' which provides much of the handle's basic functionality.
