Detailed Description

template<class _Vt, class _Et = size_t>
class cgAutoPtr< _Vt, _Et >

Assignment operator designed to assign data from another cgAutoPtr. In this case we simply reference the incoming cgAutoPtr::_data item and increment its reference count. This essentially provides support for a deferred copy of the data, where this instance of the cgAutoPtr will make itself "unique" only at the last minute if necessary.
