virtual void cgScriptInterop::DisposableScriptObject::dispose ( bool  disposeBase)
inlinevirtual

Release any memory, references or resources allocated by this object. Derived classes should implement this behavior.

This method is usually called automatically by the system either when the object is physically deleted, or when no references remain except those maintained by the scripting system. Calls can be made to this method directly however if you wish the object to dispose of its data and clean up prior to either of these circumstances occuring.

Manually calling this method can also be useful in cases where early disposal may be necessary in order to re-initialize the object's members with alternate data; in effect simulating the process of object destruction without physically deleting the object and ensures that existing references to the object remain valid.

In such situations the caller should take care to specify 'false' to the 'bDisposeBase ' parameter.

Parameters
bDisposeBaseCall the base class' 'dispose()' method upon completion.
