cgSurfaceShader Class Reference

Wrapper about the ISurfaceShader scriptable interface that provides support for describing, compiling and applying vertex and pixel shader permutations, DirectX effect technique like script method access, and configurable / dynamic material property definitions. This is effectively Carbon's portable and extended effect file architecture analogue. More...

Detailed Description

Wrapper about the ISurfaceShader scriptable interface that provides support for describing, compiling and applying vertex and pixel shader permutations, DirectX effect technique like script method access, and configurable / dynamic material property definitions. This is effectively Carbon's portable and extended effect file architecture analogue.
