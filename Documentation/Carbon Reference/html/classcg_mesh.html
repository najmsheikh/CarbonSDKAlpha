 Begin preparing the mesh. This consists of setting a vertex source and adding primitives to the relevant buffers through the 'AddPrimitive' method. Once preparation is complete, call 'endPrepare' to allow the rendering buffers to be built. Note : Calling prepareMesh will cause any existing data to be cleared. The caller can however roll back an earlier call to 'endPrepare' by passing a value of true to the 'bRollBackPrepare' parameter.
